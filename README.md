# Driver Drowsiness Detection

Đồ án AI cơ bản và ứng dụng: phát hiện buồn ngủ của tài xế theo thời gian thực bằng học sâu.

## Chức năng
- Huấn luyện mô hình phân loại Drowsy / Non Drowsy
- Dự đoán thời gian thực từ webcam
- Cảnh báo khi phát hiện buồn ngủ

## Cấu trúc
- data/: dữ liệu
- src/: mã nguồn
- models/: model đã train
- reports/: ảnh kết quả
